Usage:
from a terminal run `npm install` in both `ws-server` and `tic-tac-toe` directories

run `node server.js` from `ws-server` directory to start the server then run `npm run dev` in `tic-tac-toe` to start the client.

Open up 2 browsers off of the port given from `npm run dev` and click the squares to play tic-tac-toe!

To reset, restart both the server and the clients
